# Budgeting App (.NET Project)

This application is part of a .NET Programming subject project.

## Purpose

This app is created to help users manage their personal finances using the **50/30/20 budgeting rule** (dividing income into 50% Needs, 30% Wants, and 20% Savings).

## Features

* **Expense Management:** Users can add, edit, and delete their expenses.
* **Categories:** Assign expenses to different categories (e.g., Needs, Wants, Savings).
* **Data Import/Export:**
    * Import expenses from a .CSV file.
    * Export the current expense list to a .CSV file.
